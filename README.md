# 🧙‍♂️ Desafio DIO - Escrevendo as Classes de um Jogo

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte do desafio **"Escrevendo as Classes de um Jogo"** da plataforma **DIO (Digital Innovation One)**.

O objetivo do desafio é colocar em prática os conceitos fundamentais da programação orientada a objetos utilizando a linguagem **C#**, criando uma classe que representa um herói de uma aventura e implementando diferentes comportamentos de ataque de acordo com o tipo do personagem.

---

## 🎯 Objetivo

Desenvolver uma classe genérica que represente um herói contendo as seguintes propriedades:

* Nome
* Idade
* Tipo (Mago, Guerreiro, Monge ou Ninja)

Além disso, implementar um método `Atacar()` que exibe uma mensagem diferente conforme o tipo do herói.

### Exemplo de saída

```text
O mago atacou usando magia
O guerreiro atacou usando espada
O monge atacou usando artes marciais
O ninja atacou usando shuriken
```
